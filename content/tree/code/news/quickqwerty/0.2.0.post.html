Version 0.2.0 of QuickQWERTY is now available. The number row has been adjusted so that keys 1 to 5 are now typed with the left hand, and keys 6 to 0 with the right hand. In addition, the minimum speed requirement for moving to the next lesson has been removed, giving learners greater flexibility in their practice.

These changes make QuickQWERTY more consistent with standard typing conventions and more adaptable to different learning speeds.
