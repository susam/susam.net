Version 1.0.0 of QuickQWERTY is now available. This is the first release of this touch typing tutor on GitHub. With this release, source code and release management has been moved from SVN and SourceForge to Git and GitHub. You can now run QuickQWERTY here: quickqwerty.html.

Apart from changes in source code and release management, this release introduces input commands like restart and rst to quickly restart the current lesson. Further, in this release, the licence was changed from the GNU General Public Licence, version 3 to the Simplified BSD Licence (BSD-2-Clause).

With these updates, QuickQWERTY reaches its first major release. The source code of QuickQWERTY is now available on GitHub at github.com/susam/quickqwerty.
