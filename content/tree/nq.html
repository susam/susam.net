About Nerd Quiz

Introduction

Hi! I am Susam Pal, the developer and maintainer of Nerd Quiz.

Every question in this quiz comes from my own everyday moments of reading, writing, thinking, learning and exploring. Sometimes, when I come across something curious or fascinating, I turn it into a quiz question and share it here. Each question includes a footer that shows when I wrote the question as well as the penalty score if you answer it incorrectly.

New quizzes are published whenever I have five fresh questions ready. There's no fixed schedule.

URL Tricks

The hash value in the URL determines the quiz presented to you. If it is an integer like #1, #2, etc., then the corresponding quiz with a set of five questions is loaded. However, if the hash value starts with the prefix "q", like #q1, #q2, etc., then a specific question is loaded. You can also mix and match questions to form a custom quiz. To do so, use the prefix "q" followed by comma-separated integers as the hash value, e.g., #q1,3,5.

This is a very new project, so currently there is only one quiz. As more quizzes are added, a few navigation links will be added too to make it easy to to explore the collection, jump to a random quiz or generate a custom set of questions on the fly.
